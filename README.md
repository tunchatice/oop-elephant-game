# Java OOP Elephant Game

## Overview

This project is a Java-based object-oriented game developed as a course project. It focuses on game logic, class-based design, state management, and interactive gameplay mechanics.

The main goal of the project is to model a playable game structure using core object-oriented programming principles such as abstraction, encapsulation, class responsibility separation, and reusable logic.

## Features

- Object-oriented game structure
- Separate classes for core game components
- Interactive gameplay flow
- State and rule management
- Use of Java for implementation and organization of game logic
